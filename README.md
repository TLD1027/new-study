知识宫殿
========

Java + MySQL 图书管理系统


#### 用户：
		* 修改密码
		* 个人信息查询
		* 还书
		* 图书借阅与续借
#### 管理员：
		* 修改密码
		* 录入新书
		* 更新书库信息
		* 创建新的书籍类型


### 数据库连接相关
```
private static String username = "root";
private static String userpassword = "root";
private static String url = 
"jdbc:mysql://localhost:3306/my_library";
```
